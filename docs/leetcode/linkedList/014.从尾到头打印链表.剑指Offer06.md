# 从尾到头打印链表（剑指Offer 06）

## 题目

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

## 示例

输入：`head = [1,3,2]`
输出：`[2,3,1]`

## 限制

- 0 <= 链表长度 <= 10000

## 算法

### reverse()

```js
export const reversePrint = (head) => {
	const arr = [];
	while (head) {
		arr.push(head.val);
		head = head.next;
	}
	return arr.reverse();
};
```

### 递归

注意`push()`返回的是操作后数组的长度`length`，不能直接`return`

```js
export const reversePrint = (head) => {
	if (!head) return [];
	let arr = reversePrint(head.next);
	arr.push(head.val);
	return arr;
};
```
